yang digunakan untuk mengelola data quotes menggunakan PHP dan JavaScript.

## Teknologi yang Digunakan
- HTML
- CSS
- JavaScript (Fetch API)
- PHP (Native)
- MySQL

## Cara Menjalankan Aplikasi
1. Pindahkan folder `ujian` ke dalam folder `htdocs` (XAMPP).
2. Jalankan Apache dan MySQL.
3. Import database ke phpMyAdmin.
4. Buka browser dan akses:

http://localhost/ujian/public/index.html
